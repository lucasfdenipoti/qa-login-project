# Casos de Teste

---

# CT-01 — Login com sucesso

## Objetivo
Validar login com credenciais válidas.

## Passos

1. Acessar tela de login
2. Inserir e-mail válido
3. Inserir senha válida
4. Clicar em Entrar

## Resultado Esperado

Usuário autenticado com sucesso e redirecionado para dashboard.

---

# CT-02 — Login com senha inválida

## Objetivo
Validar comportamento ao inserir senha inválida.

## Passos

1. Acessar login
2. Inserir e-mail válido
3. Inserir senha inválida
4. Clicar em Entrar

## Resultado Esperado

Sistema deve exibir “Usuário ou senha inválidos”.
