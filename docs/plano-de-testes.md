# Plano de Testes

## Objetivo

Validar o funcionamento da funcionalidade de login da plataforma.

---

## Escopo

### Dentro do escopo

- Login com credenciais válidas
- Login com senha inválida
- Campos obrigatórios
- Mensagens de erro

### Fora do escopo

- Cadastro de usuário
- Recuperação de senha
- Integrações externas

---

## Tipos de Teste

- Teste funcional
- Teste exploratório
- Teste de usabilidade

---

## Ambiente

- Navegador: Google Chrome
- Sistema Operacional: Windows 11
- Ferramenta de gestão: Jira
