# User Story — Login

## História de Usuário

Como usuário da plataforma,  
quero realizar login utilizando e-mail e senha válidos,  
para acessar minha conta e utilizar as funcionalidades do sistema.

---

## Critérios de Aceite

- O sistema deve permitir login com credenciais válidas.
- O sistema deve exibir mensagem de erro para credenciais inválidas.
- O sistema deve validar campos obrigatórios.
- O sistema deve redirecionar o usuário para a dashboard após login válido.
